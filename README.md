# Simulador de Cajero Automático en Python

Este proyecto es una simulación de un cajero automático (ATM) desarrollado en Python.  
Incluye operaciones básicas como consultar saldo, depositar, retirar y validar entradas.

#Funcionalidades
- Consultar saldo actual.
- Depositar dinero con validación.
- Retirar dinero verificando fondos disponibles.
- Manejo de errores con `try/except`.
- Menú interactivo en consola.
- Lógica de negocio implementada paso a paso.

#Estructura del Proyecto
- `main()` controla el flujo del programa.
- Funciones separadas para cada operación:
  - `consultar_saldo()`
  - `depositar()`
  - `retirar()`

#Tecnologías usadas
- Python 3
- Manejo de errores
- Programación estructurada
